Engineering Design and Thinking
Prototype Mover Robot

A year long class where we tailored a LEGO EV3 robot in Python to be a part of a mobile processing plant that replicated the movements of an ant, enabling it to identify various materials in disasterous areas and sort them into designated bins so it can be covert debris into renewable energy source. The robot utilized Proportional-Integral-Derivative (PID) control to navigate challenging terrains and accurately follow both continuous and dashed curved paths using sensors and actuators. Our team was among the few to successfully reach the final track of 30m in both semesters.

Database Design and Analysis
Generic Shopping Application

In this class, we applied our theoretical knowledge of databases to develop a generic shopping application using PostgreSQL and Flask that includes a login page, items with pictures and prices, search button to browse and filter items by category, adding the items or remove them from their shopping cart and place orders. All of which updates synchronously on the database. Additionally, we designed a separate interface for employees to manage inventory in warehouses, edit product information, view orders, and perform other administrative tasks by logging into their portals.

Computer Networks
Public Bulletin Board System

Through this class, I implemented a client-server bulletin board system using raw socket programming in Python. utilizing its threading library for each its users processes. I created a public message board where users could connect to a server and join a public group. After joining, users can post and join multiple private groups. The server kept track of all connected users, notifying everyone when a user joined or left. Users can retrieve messages through unique messageID and the messages of a private group can only be visible to its users. Users can only view the last two messages sent in the group to avoid congestion.

Programming Languages
STIMPL

In this class was to develop a Turing-complete imperative programming language called STIMPL that extends Python. We were provided with a shell code to implement the evaluation of various expressions such as literals, variable assignments, mathematical operations, logical conditions, and control flow constructs. On top of that, I managed a program state using a 'State' class to track variable types and values. The implementation required handling error cases, raising exceptions for type mismatches and syntax violations.
